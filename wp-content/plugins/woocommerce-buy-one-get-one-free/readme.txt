Create "Buy One, Get One Free" deals in your WooCommerce store.

== Installation ==

1. Download, install and activate the plugin.
1. Go to WooCommerce > Settings > Products > Buy One Get One Free to set plugin options.
1. Go to WooCommerce > Buy One Get One Free to create BOGO rules.
